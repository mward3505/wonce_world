# Wonce World 🌍

A fun, bilingual (English/Spanish) kids learning app built with React Native & Expo.

Children can learn letters, numbers, colors, and shapes through flashcards, quizzes, and a color-hop mini game — all behind a parent-protected PIN.

## Features

- 👶 Multiple kid profiles
- 🔤 Letters, 🔢 Numbers, 🎨 Colors, 🔷 Shapes
- 🎮 Color Hop mini game
- 📝 Quick quizzes
- 🔊 Audio read-aloud on flashcard tap
- 🇺🇸 / 🇪🇸 English & Spanish
- 🔒 Parent area protected by PIN

---

## Run Locally

**Prerequisites:** Node.js, npm, [Expo Go](https://expo.dev/go) on your phone

```bash
git clone https://github.com/mward3505/warpos-world.git
cd warpos-world
npm install
npx expo start
```

Scan the QR code in the terminal with Expo Go.
